Intro
#####

Linux is a unix-like operating system that had its first release in 1991. See `History of Linux <https://en.wikipedia.org/wiki/History_of_Linux>`_.

It is an Open-Source operating system.

There are `many distributions <https://en.wikipedia.org/wiki/List_of_Linux_distributions>`_

Distributions like Android, Debian, Ubuntu and ChromeOS are widely used.

Linux is being used to run most cloud services and IOT/embedded devices.
