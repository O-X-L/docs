============
Docs Toolset
============

Tools used by the Documentation
*******************************

* The text **was NOT AI generated**. All made-by-human.

  * We only use `deepl.com <https://www.deepl.com/en/translator>`_ to speed-up content translation (EN=>DE)

* The documentation is written in `reStructuredText <https://www.sphinx-doc.org/en/master/usage/restructuredtext/basics.html>`_ and converted to HTML by `Sphinx <https://www.sphinx-doc.org>`_

  * Source repository: `github.com/O-X-L/docs <https://github.com/O-X-L/docs>`_

* The modern styling is provided by `github.com/piccolo-orm/piccolo_theme <https://github.com/piccolo-orm/piccolo_theme>`_

* The documentation is served using `nginx <https://nginx.org/en/>`_, proxied using `HAProxy <https://www.haproxy.com/>`_, firewalled using `NFTables <https://netfilter.org/projects/nftables/>`_ and running on `Proxmox VE <https://www.proxmox.com/en/proxmox-virtual-environment/overview>`_.

* Charts are created using `lucid.app <https://lucid.app>`_ (alternatively `draw.io <https://draw.io>`_)
