Intro
#####

Linux ist ein unixähnliches Betriebssystem, das erstmals 1991 veröffentlicht wurde. Siehe `Geschichte von Linux <https://en.wikipedia.org/wiki/History_of_Linux>`_.

Es ist ein Open-Source-Betriebssystem.

Es gibt `viele Distributionen <https://en.wikipedia.org/wiki/List_of_Linux_distributions>`_ welche die auf den Grundfunktionalitäten von Linux aufbauen.

Distributionen wie Android, Debian, Ubuntu und ChromeOS sind weit verbreitet.

Linux wird für die meisten Cloud-Dienste und IOT/eingebettete Geräte verwendet.
